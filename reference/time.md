# djta

- djta: djnisi tamo = rotate time
- djni-si: turn-is = rotate

- djta = day
- djtasi = today
- djtafu = yesterday
- djtafure = the day before yesterday
- djtaha = tomorrow
- djtaharj = the day after tomorrow

- djtati: djta-[ti](/section/basic-word#number) = day seven = week
- djtatilo = weekend
- djtatiha = weekday

## example

- si ya buti djta!
- it is beautiful day

- djtafu sjma ya!
- yesterday similar is!
