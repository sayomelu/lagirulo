# djta

- djta: djniso tamo = rotate time
- djni-so: turn-is = rotate

- djta = day
- djtaso = today
- djtafu = yesterday
- djtafure = the day before yesterday
- djtapa = tomorrow
- djtaparj = the day after tomorrow

- djtati: djta-[ti](/section/basic-word#number) = day seven = week
- djtatilo = weekend
- djtatiha = weekday

## example

- so ya buti djta!
- it is beautiful day

- djtafu sjma ya!
- yesterday similar is!
